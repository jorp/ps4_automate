# ps4_automate

## usage
```bash
$ python run_macro.py <macro input file>
```

### Example
The included `example.txt` would map to the following:
```text
3lu		--> 3x left-stick up
rl		--> 1x right-stick left
rr		--> 1x right-stick right
rr		--> 1x right-stick right
rl		--> 1x right-stick left
2t		--> 2x triange
r2		--> 1x R2
```
